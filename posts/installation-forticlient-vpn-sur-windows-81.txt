Contournement
-------------

* Désinstaller le forticlient
* Supprimer les traces résiduelles (ie, connexion vpn...)
* Re-télécharger le `forticlient VPN <http://www.forticlient.com>`_
* Créér un utilisateur local avec droit **admin local**
* Se déconnecter - Se reconnecter avec cet utilisateur admin local
* Réinstaller le forticlient VPN.
